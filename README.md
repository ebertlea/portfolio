## Projects

### Contrastive Representation Learning
**Repo:** [contrastive-representation-learning](https://github.com/ebertlea/contrastive-representation-learning)

SimCLR-style contrastive learning on MNIST, building on my MSc thesis work — a PyTorch pipeline for analysing COVID antibody test images where labelled data is scarce. The goal: learn meaningful feature representations without manual labels.

- CNN encoder + MLP projection head trained with NT-Xent loss
- Augmentation pipeline (crop, flip, colour jitter, Gaussian blur)
- Linear evaluation benchmark to measure representation quality
- Built with PyTorch
